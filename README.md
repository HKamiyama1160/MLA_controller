# MLA_controller
 MLA(Magnetic Loop Anttena) controller by Wifi module.  
 ESPr-Developer module is used , made by SwitchSceince.  
 Motor driver , TOSHIBA TA7291P had been discontinued , but compatible module is available.  
  https://www.switch-science.com/products/5880?_pos=7&_sid=366f102be&_ss=r
 because ESP IP address is got by DHCP, "IP-scanner"(Windows),   
  "nmap"(Linux), other application available to connect ESP module.
